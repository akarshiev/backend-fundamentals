# Dynamic va Static IP

## Dynamic IP

ISP sizga vaqtinchalik IP beradi.

Bugun: `84.54.12.8`

Ertaga: `84.54.12.50`

bo'lishi mumkin.

Bu: **Dynamic IP** deyiladi.

Ko'pchilik uy internetida shunday.

---

## Static IP

Doim bir xil.

Misol: `84.54.12.8` har doim sizniki.

Serverlar odatda static IP ishlatadi.

---

## Dynamic vs Static

| Xususiyat | Dynamic | Static |
|-----------|---------|--------|
| IP o'zgarishi | Ha, vaqtinchalik | Yo'q, doimiy |
| Narxi | Arzonroq | Qimmatroq |
| Qo'llanilishi | Uy interneti | Serverlar |
| ISP | Avtomatik beradi | So'rash kerak |

---

## Dynamic IP muammolari

- DNS record yangilash kerak
- Port forwarding ishlamasligi mumkin
- VPN ulanishda qiyinchilik

---

## DDNS yechimi

Dynamic DNS - IP o'zgarsa ham domain nomi ishlayveradi.

Mashhur DDNS xizmatlari:

- No-IP
- Dynu
- DuckDNS
